# 🎓 Campus Pulse – Student Satisfaction Dashboard

## 📌 Project Overview

Campus Pulse is an interactive data analytics dashboard developed to analyze student satisfaction across various campus facilities such as cafeteria, hostel, library, sports center, and Wi-Fi.
The system enables real-time filtering, visualization, and insights generation to support data-driven decision-making for improving campus services.

---

## 🎯 Objectives

* Analyze student feedback efficiently
* Identify satisfaction trends across departments and facilities
* Provide insights for improving campus infrastructure
* Build an interactive data-driven dashboard using Python

---

## 🚀 Features

* 🔐 Secure Login System
* 📊 Interactive Dashboard using Streamlit
* 🔎 Dynamic Filters (Department, Year, Facility)
* 📈 Visualizations (Bar Chart, Pie Chart, Line Chart, Box Plot)
* 🗄️ SQL Integration using SQLite
* 📥 Download Filtered Data (CSV)
* 📢 Automated Insights Generation

---

## 🛠️ Technologies Used

Programming Language: Python
Data Analysis: Pandas
Visualization: Matplotlib, Seaborn
Dashboard Framework: Streamlit
Database: SQLite (SQL)
UI Styling: Streamlit + basic CSS

---

## 📂 Dataset

* Data collected using Google Forms
* Exported into CSV format
* Dataset fields include:

  * Student ID
  * Department
  * Year
  * Facility
  * Rating (1–5 scale)
  * Feedback Date

---

## ⚙️ Workflow

1. Data Collection using Google Forms
2. Data Preprocessing using Python (Pandas)
3. Data Storage in SQLite Database
4. Data Analysis using SQL and Pandas
5. Data Visualization using Matplotlib and Seaborn
6. Dashboard Development using Streamlit
7. Deployment using Streamlit Cloud

---

## 📊 Key Insights

* Department-wise satisfaction comparison
* Facility performance evaluation
* Identification of low-performing services
* Trend analysis of student feedback over time
